# 🏏 IPL Data Analysis Project

## 📌 Overview

This project analyzes Indian Premier League (IPL) data using Python.
The goal is to extract meaningful insights about players and teams using data analysis and visualization techniques.



## 📊 Features

* 🟠 Orange Cap (Overall & Season-wise)
* 🟣 Purple Cap (Overall & Season-wise)
* 🏏 Top Batsmen Analysis
* 🎯 Top Bowlers Analysis
* 📈 Strike Rate & Economy Rate
* 🏆 Most Successful Teams
* 🔥 Most Consistent Batsmen
* 📊 Data Visualization using Matplotlib



## 🧠 Key Insights

* Identified top run-scorers and wicket-takers across seasons
* Found most consistent batsmen based on average runs per match
* Analyzed team dominance based on total wins
* Visualized trends of player performance over different IPL seasons



## 📊 Sample Insights

* 🟠 Highest runs in a single season: Virat Kohli (973 runs in 2016)
* 🟣 Highest wickets in a season: Dwayne Bravo (32 wickets in 2013)
* 🔥 Most consistent batsman: KL Rahul (highest average runs per match)
* 🏆 Most successful team: Mumbai Indians (highest match wins)

These insights were derived using data analysis techniques such as grouping, aggregation, and filtering.



## 🛠️ Technologies Used
